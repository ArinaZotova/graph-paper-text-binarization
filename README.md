# Graph Paper Text Binarization

Курсовая работа по дисциплине **«Методы и средства обработки изображений»**.

Проект реализует алгоритм бинаризации рукописного текста на изображениях графической бумаги. Алгоритм подавляет клетчатую структуру листа и формирует бинарную маску текста.

## Описание задачи

На вход подается изображение рукописного текста на листе графической бумаги. Необходимо получить бинарную маску, в которой белым цветом выделен только рукописный текст, а фон и линии клетки остаются черными.

Для оценки качества может использоваться эталонная маска, размеченная вручную.

## Используемые технологии

- C++17;
- OpenCV;
- CMake.

## Основные этапы алгоритма

1. Загрузка исходного изображения.
2. Предварительная обработка: обрезка рабочей области и приведение к единому размеру.
3. Перевод изображения в градации серого.
4. Применение быстрого преобразования Фурье.
5. Подавление частот, соответствующих клетчатой структуре.
6. Обратное преобразование Фурье.
7. Бинаризация методом Оцу.
8. Дилатация и очистка связных компонент.
9. Расчет метрик качества при наличии эталонной маски.

## Метрики качества

В проекте рассчитываются следующие метрики:

- IoU;
- Precision;
- Recall;
- F1-score.

Для итогового эксперимента на реальном изображении были получены следующие значения:

| IoU | Precision | Recall | F1-score |
|---:|---:|---:|---:|
| 0.6532 | 0.9012 | 0.7037 | 0.7903 |
